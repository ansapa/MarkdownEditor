# MarkdownEditor

Test (updated Github pwd)

MarkdownEditor is an experiment to create a SwiftUI editor with syntax highlighting for Markdown. It includes my own Markdown parser, which tries to be Commonmark compliant (currently only 23% of the test cases pass).

<img width="100%" alt="Screenshot" src="https://user-images.githubusercontent.com/50514472/114135827-1b334680-990a-11eb-81b0-4776f389c682.png">

Currently supported:

- Thematic breaks
- ATX Headings
- Setext Headings
- Intended code blocks
- Fenced code blocks:
- Paragraphs
- Block quotes
- Lists
